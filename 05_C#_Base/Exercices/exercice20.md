# Exercice 20

Écrire une fonction **CalculerResultat** qui prend en paramètres 3 double (les notes) et retourne un tuple (double moyenne, string mention).

Règles de mentions :

Moyenne < 10 → "Échec"
10 ≤ Moyenne < 12 → "Passable"
12 ≤ Moyenne < 14 → "Assez bien"
14 ≤ Moyenne < 16 → "Bien"
16 ≤ Moyenne ≤ 20 → "Très bien"

Ensuite, dans Main, nous demanderons les 3 notes à l’utilisateur.
Appelez ensuite la fonction, et affiche le résultat.
