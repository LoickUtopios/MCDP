# Exercice 21

- Écrire un programme se servant d'une fonction nommée `operation` retournant, à partir de deux nombres en paramètres, un tuple contenant :
    - La somme
    - La difference
    - Le quotient
    - Le produit

- Vous testerez cette fonction dans le cadre d'un programme console
demandant à l'utilisateur les deux valeurs.

- Enfin, afficher les résultats reçu par le tuple.
